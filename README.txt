HORSEHORSE
================================================================================
HorseHorse is an tank-control duel-ish game, where valiant non-horses are pitted
against one another.
Made for Librejam 2020-02, with the theme "SUBNAUTICAL."


RUNNING
--------------------------------------------------
LINUX/UNIX/HAIKU
--------------------
Make sure love2d is installed with your package manager, and then:
	$ make test
